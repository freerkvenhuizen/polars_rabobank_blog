# Examples supporting my blog post on how Rabobank is using Polars
This repo contains example code on how we use Polars to detect periodic payment bookings

# How to use?

## Install
- Make sure you have Poetry installed (https://python-poetry.org/docs/)
- Run 'poetry install', it will create a virtual environment with all required dependencies
- Activate the environment, i.e., run 'poetry env activate'
- Run 'poetry run pytest tests' to run all unit tests and performance tests
